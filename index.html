projects

Here are some projects that I have worked on that help to showcase my skills as a developer.

Enigma Machine

Programmed in Python, my Engima machine was an attempt at re-creating the encryption algorithm used by the Germans during the second World War. The original machines used to encrypt messages involved many moving parts that were physically changed to set configurations.

The program I wrote allows the user to configure their machine using the command line which allows for a unique encryption and decryption cipher.

Check it out!

Hurtle

Hurtle is a Wordle-like game designed to be a little bit more challenging than the New York Time's daily game. Hurtle has the user guess words without knowing the length of the word and without showing a keyboard of possible characters.

This game was programmed with JavaScript and was an exercise in UI/UX design, web programming, and algorithms.

Check it out!

Personal Website

My personal website was built to hub my social and professional links and showcase the programming projects that I am most proud of. This website is almost entirely static, and requires no backend. This was more of an exercise in UI/UX design.

You're here right now.

"UVA Events" Event Planner

"UVA" Events is a Django app developed by myself and a small group of students at the University of Virginia built for fun that allows students to create, plan, and advertise events around grounds.

This app was deployed on Heroku and has a fully functional backend and commands available to superusers to manipulate user postings.

Check it out!

Additional credits: Manmeet Singh, Henry Brorsen, Niroj Paraluri, Ciel Park

Wahoo-What Major?

Wahoo-What Major was an app developed by myself and another student designed to help new students at the University of Virginia decide on a major based on how much or little they agreed with 26 different questions that revealed something about their career interests.

This app was primarily built using a PHP backend running on an Apache server.

Check it out!
